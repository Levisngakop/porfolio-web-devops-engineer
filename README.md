# Levis DevOps Portfolio 

Welcome to my personal DevOps Portfolio website — built using [Bolt.new](https://bolt.new), React, TailwindCSS, and hosted on Netlify.

This site showcases my journey, skills, and projects as I pursue. As a passionate DevOps and Cloud engineer, I aspire to share my experience while learning from others as well.

##  About This Project

This portfolio is a work-in-progress and evolves as I grow. It highlights:

-  Who I am — a passionate DevOps Engineer with 5+ years of experience  
-  Tools & Technologies I work with (AWS, Docker, GitHub Actions, ArgoCD, Terraform, and more)  
-  Real-world DevOps projects (VPC builds, CI/CD pipelines, EKS clusters...)  
-  Blog links and insights from my Medium posts

##  Tech Stack

- **Frontend**: React, Tailwind CSS, Bolt.new
- **Version Control**: Git & GitHub
- **Deployment**: Netlify
- **Markdown-friendly** README for open source clarity
